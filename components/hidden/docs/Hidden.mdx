## Usage

Use to hide content either visually or from screen readers.

## Visually Hidden

This is useful when needing to hide content from the screen, but make it available screen readers. A primary use case for this is when using icon buttons; providing the icon button action as visually hidden and the icon itself as hidden only from screen readers.

<Playground data-test="visually-hidden">
  <SpacedGroup>
    <Hidden>I am hidden</Hidden>
    <span>I am not hidden</span>
    <Hidden>
      <div>I am also not visually displayed</div>
    </Hidden>
    <div>I am also not visually hidden</div>
  </SpacedGroup>
</Playground>
