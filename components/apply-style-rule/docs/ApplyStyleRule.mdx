## Usage

Apply a style rule directly to children without rendering a parent element.

## Examples

<Playground data-test="examples">
{()=>{
const styleRule = ({color}) => ({color});
return (
  <Fragment>
    <ApplyStyleRule color="coral" styleRule={styleRule}>I am coral.</ApplyStyleRule>
    <ApplyStyleRule color="purple" styleRule={styleRule}>
      <div>I am purple.</div>
    </ApplyStyleRule>
</Fragment>
);
}}

</Playground>
